# Tools

> 实现从Harbor拉取镜像、保存镜像到本地等功能。

### `docker_tools`

> `images.txt`每行为标准镜像名称和版本号。

执行命令：

```bash
./docker_tools

# 拉取镜像选择操作2即可

# ==============================================
# Docker 镜像批量操作工具
# ==============================================
# 1. 批量推送镜像到 harbor.vastaitech.com/ai_deliver
# 2. 批量从 harbor.vastaitech.com/ai_deliver 拉取镜像（智能回退）
# 3. 批量保存镜像到本地
# 4. 批量从本地加载镜像
# 5. 显示镜像列表文件内容
# 6. 批量删除本地镜像
# 7. 退出
# ==============================================
# 请选择操作 (1-7): 
```
